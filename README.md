 Number Guessing Game (Java)

 📌 Project Description

This is a simple console-based Number Guessing Game built using Java.
The computer randomly selects a number between 1 and 100, and the user tries to guess it.
The program gives hints like "Too High" or "Too Low" until the correct number is guessed.

 🛠 Technologies Used

* Java
* VS Code

 ▶️ How to Run the Project

1. Clone the repository.
2. Open the project in VS Code.
3. Compile the program using:
   javac Main.java
4. Run the program using:
   java Main

 🎯 Features

* Random number generation
* User input handling
* Loop until the correct guess
* Hint system (High / Low)

 📚 Concepts Used

* Scanner class
* Random class
* Loops
* Conditional statements
* Basic Java programming
